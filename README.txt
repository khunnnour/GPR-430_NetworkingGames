Description:

A small game where you roll around a grid trying to turn the most tiles your color in the allotted time (20sec, currently) using WASD to move.



Instructions:

- Make sure a server instance is running 
  - Click the "Start Server" button in the dialgoue on start-up
- Enter IP of server (if left blank it will assume localhost 127.0.0.1)
- Click "Start Client"
- Wait for clients to join, then on one client click the "Start" button in the bottom left corner
